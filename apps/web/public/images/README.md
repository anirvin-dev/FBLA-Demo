# Yolo Logo Assets

Please place your Yolo logo images in this directory:

1. **`logo-yolo-light.png`** - Light/outline version (for use on light backgrounds)
2. **`logo-yolo-dark.png`** - Dark/filled version (for use on dark backgrounds)

These images should be the cute cartoon animals (otter/seal) hugging that represent the Yolo brand.

Recommended dimensions: 512x512px or higher (will be displayed at various sizes throughout the app)

## Other Business Images
